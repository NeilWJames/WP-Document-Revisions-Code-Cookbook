Add-on to WP Document Revisions to support WPML.

== Description ==

This plugin is an add-on to WP Document Revisions to support WPML.

See [addition information](./readme.md) for how to use it.

== Installation ==

1. Download the Cookbook code from GitHub
2. Copy the `wpdr-wpml-support` directory to your plugins directory. This can be done by creating a zip file of the directory and loading that as a plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress
