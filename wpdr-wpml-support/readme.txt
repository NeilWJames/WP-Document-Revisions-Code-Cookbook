Add-on to WP Document Revisions to support WPML.

== Description ==

This plugin is an add-on to WP Document Revisions to support WPML.

See [addition information](./readme.md) for how to use it.

== Installation ==

1. Download directory wpdr-wpml-support from within the WPDR CookBook
2. Extract to `/wp-content/plugins/wpdr-wpml-support` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
